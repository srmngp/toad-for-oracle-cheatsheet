Toad for Oracle: Essential Tips & Tricks Cheat Sheet

üîç Toad Basics & Navigation

  • Schema Browser: Gateway to DB objects.
    • Display: Treeview, Dropdown, Tabbed.
    • Filtering: Quick filter (e.g., C*; D*), Project filters.
    • Organize: Favorites, Project Manager (Add objects via right-click or drag/drop).
    • Actions: Right-click for >35 DB operations.
    • Load Schemas: Right-click schema > Load only schemas that own objects.
  • Navigation:
    • Ctrl+J for Jump to.
    • Back/Forward buttons for browsing history.
    • Ctrl+Tab to cycle through child windows/tabs.
  • Multi-Select: Select multiple objects then right-click for batch operations.
  • Context Matters: Where you right-click determines menu options (grids, toolbars, headers).
  • Additional Details: Right-click column header to add Num Rows, Last Analyzed, etc.

‚å®Ô∏è SQL & PL/SQL Editor

  • Single Editor: For SQL, PL/SQL, anonymous blocks, SQL*Plus, RMAN, Java stored procedures.
  • Code Templates: Type anon then <Ctrl+Spacebar> (dynamic with &). Empty cursor shows all templates.
  • Code Insight: Auto-complete commands & objects (Ctrl+.).
    • Types: Tables, Views, Aliases, Functions, Sequences, Users, Collection types, Synonyms.
    • Sees variables, parameters, expected tokens (keywords).
    • Adjust delay in Options > Editor > Code Assist.
  • DESC (F4): View table/view definitions. Can DESC multiple objects, drag columns to editor, modify objects.
  • SQL Recall (F8): Access last 500 executed statements (Alt+Up/Down to cycle).
  • Named SQL: Ctrl+N to name statements for easy recall later.
  • Split Editor: Right-click in editor > Split Editor Layout (Top/Bottom or Left/Right).
  • Format Code: Right-click > Format Code to beautify SQL/PL/SQL.
  • Make/Strip Code: Ctrl+M to make code statement (C++, Delphi, Perl, Java, VB, C#), Ctrl+P to strip.
  • Other Keys:
    • Ctrl+F9: Verify statement without execution (parse only).
    • Ctrl+F12: Pass SQL to external editor.
    • Ctrl+E: Execute Explain Plan on current statement.
    • F2: Toggle full screen Editor.
    • F5: Execute as script.
    • F6: Toggle between Editor and Results panel.

üêû PL/SQL Debugging & Performance

  • Profiler: Identify execution bottlenecks.
    • Toggle Stopwatch button, then execute PL/SQL.
    • View visual charts in Profiler tab (Details button).
    • Supports Oracle 11g+ Hierarchical Profiler.
  • Debugger: Step-by-step code execution.
    • Breakpoints: Shift+F5 set/delete on executable lines (blue dots only).
    • Conditional Breakpoints: Double-click break in Breaks panel.
    • Watches: View variable values (eyeglasses icon), Ctrl+Alt+B to add watch at cursor.
    • Smart Watches: Enable to see all code variables automatically.
    • Modify: Change variable values (calculator icon).
  • Debugger Shortcuts:
    • F11: Run (continue execution).
    • F12: Run to cursor.
    • Shift+F7: Trace into.
    • Shift+F8: Step over.
    • Shift+F10: Trace out.
    • Ctrl+F5: Display Breakpoints window.
    • Ctrl+Alt+W: Display Watches window.
    • Ctrl+Alt+C: Display Call Stack.
    • Ctrl+Alt+E: Display Evaluate/Modify window.
    • Shift+Ctrl+F9: Set parameters.
  • Run SQL in PL/SQL: Ctrl+Enter on embedded SQL (Toad extracts and prompts for variables).

üìä Data Grid Mastery

  • Sorting/Filtering:
    • Click cell for Add to filter... current value.
    • Use Sort panel for multiple conditions, control NULL handling.
    • Named filters can be saved and recalled.
    • Manual WHERE clause input supported.
    • Red funnel icon = filter active.
  • Column Management:
    • Drag header to re-order/hide (X symbol).
    • Right-click > Reset columns.
    • Use column selector (top left) to show/hide columns.
    • Fix columns to keep visible while scrolling.
  • Grouping:
    • Right-click header > Toggle Group Panel.
    • Drag columns to group panel for hierarchical grouping.
    • Multiple levels supported.
  • Calculated Fields: Select cells, click Sigma button for Sum, Avg, Min, Max, Count (across multiple columns).
  • Grid Styles: Apply visual color styles (Options > Data Grids > Visual): Standard, Cool, Query Builder, etc.
  • Single Record View: "Open book" button for vertical record display. Navigate with arrow buttons.
  • Export/Copy:
    • Ctrl+C to copy selected cells.
    • Right-click > Export Dataset for multiple formats.
    • Column order/visibility respected in exports.
  • Reports: Right-click > Report to launch report generation wizard.

üîó Database Utilities & Modeling

  • ER Diagram: Generate visual DB relationships.
    • Select objects in Schema Browser > Right-click > ER Diagram.
    • Export to interactive HTML (WYSIWYG).
    • Based on foreign key constraints.
    • Use Toad Data Modeler (free for licensed users) for more features.
    • Infer Relationships: In Data Modeler, guess relationships from common column names.
  • Code Road Map: PL/SQL dependency analysis.
    • Right-click on trigger/procedure/function/package/view > Code Road Map.
    • Parses source code, identifies dependent objects.
    • Export to HTML report.
    • Create Script: Generate DDL to build sandbox environment with all dependencies.
  • Query Builder: Visually build complex SQL queries.
    • Drag & Drop tables/columns from Object Palette.
    • Auto-detect joins (from FKs).
    • Double-click join link to view/modify conditions.
    • Add WHERE clauses with value picker for dates.
    • Right-click columns to include in SELECT or WHERE.
    • Execute (F9), view Explain Plan.
    • Reverse-engineer: Send existing query to Query Builder.
    • Supports ANSI join syntax.
  • Jump Search (v12.0+): Search all Toad windows, menus, and options. Navigate UI quickly.

üí∞ Professional & Xpert Editions

Professional Edition:

  • Generate Test Data: Select one+ tables, automatic generation respects FKs, creates real-looking data (names, addresses, etc.).
  • Code Analysis: Automated SQL/PL/SQL code review, best practices advisor, automatic documentation generation.
  • Testing Suite:
    • Code Tester: Generate functional tests without writing PL/SQL. Run to test feature.
    • Benchmark Factory: Load test procedures. Right-click > Send to Code Tester for performance testing.
    • Unit testing, regression testing.

Xpert Edition (SQL Optimizer):

  • SQL Optimization: Identify problematic SQL, auto-tune statements, suggest indexes.
  • Optimize Current SQL: From Editor, click "Auto Optimize SQL" button. Choose OLTP/Warehouse, set search depth.
  • Advanced SQL Optimization: Full tuning lab. Rewrite & Index scenarios.
    • Optimize: Auto-rewrite + execute (fast queries).
    • Rewrite: Generate alternatives, evaluate plans manually (long queries).
    • Index: Create virtual indexes, test before physical creation.
    • Rewrite & Index: Both scenarios.
  • Compare Scenarios: View SQL syntax & execution plan differences side-by-side.
  • Optimize Indexes: Analyze SQL workload (AWR, Foglight, SGA, source code). Recommend optimal index set.
  • Impact Analysis: Model changes (new indexes), see effect on all existing SQL before implementation.
  • Plan Control (11g+): Deploy better plans (SQL Plan Baselines) without app changes. Test 60+ execution plans, deploy best.

üõ†Ô∏è DB Admin Module

  • Health Check: Comprehensive DB checkup for instances.
    • Identifies ORA-600s, unanalyzed objects, synonyms to non-existent objects.
    • 16 checks for Oracle RAC, 11 checks for production/virtualized environments.
    • Schedule checks, email results automatically.
    • Create multiple check types for different instances/days.
  • Database Browser: Effective instance management.
    • Connect to multiple databases, see parameters side-by-side.
    • Manage users, roles, tablespaces, system privileges.
    • Startup/shutdown instances.
    • RMAN integration for backups.
    • Drill down into objects like Schema Browser.
  • Statspack Browser: Analyze DB activity over time.
    • View reads/writes by tablespace over last 24 hours.
    • Top sessions analysis.
    • Visual charts (zoom, highlight, export to Excel).
    • Licensed Diagnostic Pack? Access AWR/ASH/ADDM reports.
    • Manage Statspack collection jobs and snapshots.
  • Trace File Browser: Visually inspect trace files without TKprof.
    • See all queries with binds, waits, performance profiles.
    • View Execution vs. Parse vs. Fetch vs. Wait times.
    • Click query to see bind variable values.
    • Chart showing query count by time period.

üîå Connectivity & Data Export

  • EZCONNECT: Direct DB connection without TNSNames (since 8i).

    CONNECT user/pass@[//]host[:port][/service_name]

    Example: scott/tiger@//localhost:1521/orcl

  • Building WHERE IN lists: Drag values from data grids to editor for automatic comma-delimited lists with quotes.
  • Export Data to Excel:
    • Copy/Paste: Ctrl+C / Ctrl+V. Simple, but no formatting, requires Excel open.
    • Export Dataset (Delimited Text): Fast for large data (~25 sec for big sets), but not native XLS/XLSX.
    • Export respects column order/visibility.
  • Custom Queries: Generate dynamic SQL scripts based on selected objects.
    • Select objects > Right-click > Custom Queries.
    • Example: Generate constraint inventory for multiple tables in 3 steps.
    • Create your own or modify defaults.
    • Toad ships with several pre-built (Constraint Type Summary, etc.).
  • Toad Data Point: Query non-Oracle databases (SQL Server, MySQL, DB2, Sybase, PostgreSQL, Teradata). Free for Toad Suite users.
